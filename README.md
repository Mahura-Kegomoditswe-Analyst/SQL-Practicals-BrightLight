## üß† Overview

This repository demonstrates practical SQL skills from beginner to advanced level, including:
- Data selection and filtering (`SELECT`, `WHERE`, `DISTINCT`)
- Aggregation and grouping (`COUNT`, `AVG`, `SUM`, `GROUP BY`, `HAVING`)
- Conditional logic (`CASE`)
- Handling NULL values
- Using different types of JOINs to combine tables
- Real dataset analysis with Excel visualizations and pivot tables

All practicals were executed in **Snowflake** using uploaded CSV datasets.
